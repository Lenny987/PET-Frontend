# Excel‑таблица на чистом JavaScript

Веб‑приложение, имитирующее базовый функционал Excel:  
- редактируемые ячейки (клик – ввод текста)  
- поддержка формул, начинающихся с `=` (сложение, вычитание, умножение, деление)  
- возможность сохранять/загружать данные в формате JSON  
- адаптивный интерфейс  

**Технологии:** HTML5, CSS3 (Grid/Flex), JavaScript (ES6+). Без сторонних библиотек и фреймворков.

## Как запустить

1. Скачайте папку с проектом  
2. Откройте `index.html` в любом современном браузере  
3. Готово – никаких сборщиков или зависимостей не требуется

## Функционал

- **Редактирование ячеек:** двойной клик или Enter на ячейке – ввод текста.  
- **Формулы:** `=A1+B2`, `=C3*2`, `=SUM(A1:A5)` (базовая реализация).  
- **Экспорт/импорт:** сохранение состояния таблицы в файл и загрузка обратно.  
- **Навигация:** стрелки клавиатуры, Tab для перехода между ячейками.

## Как это работает

Таблица реализована через динамическое создание сетки `div` с `contenteditable` или через `table`.  
Формулы парсятся регулярными выражениями и вычисляются с использованием объекта с данными ячеек.
