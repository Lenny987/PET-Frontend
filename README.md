# Мои frontend‑pet‑проекты

Репозиторий с учебными и личными проектами по фронтенд‑разработке.  
Каждый проект в отдельной папке с собственным описанием.

## Проекты

| Название | Технологии | Краткое описание |
|----------|------------|------------------|
| [Excel‑like таблица](./Сайт%20для%20курсовой%20по%20Бухгалтерскому%20учету/) | HTML, CSS, vanilla JS | Веб‑приложение, повторяющее функционал Excel: редактируемые ячейки, формулы, базовые операции. |
| [Лента товаров (Avito‑стажировка)](./avito-feed) | React, TypeScript, Zustand | Прототип ленты объявлений с редактированием и ИИ‑генерацией описания/цены. |

## Как запустить

Каждый проект запускается отдельно. Для большинства достаточно открыть `index.html` в браузере или выполнить:

```bash
cd имя_папки
npx serve .
